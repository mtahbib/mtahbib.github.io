Doctors Appointment

  • Tech Stack: Java FX,MySql
  • Github URL: Project Link

Features

User

  • Signup/Login: Allows users to create a new account or log into an existing one.
  • Book Appointment: Users can schedule appointments with doctors or hospitals.
    • Choose Doctor/Hospital: Enables users to select a specific doctor or hospital for their appointment.

Admin

  • Add Hospital: Admins can add new hospitals to the platform.
  • Add Doctors: Enables the addition of new doctors to the system.
  • Accept/Reject Appointment: Admins have the authority to approve or decline appointment requests.
  • Verify Payment Details: Responsible for verifying the payment details submitted by users.
  • See User Details: Admins can view detailed information about users.
